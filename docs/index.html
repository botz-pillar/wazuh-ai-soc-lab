The six phases

One continuous session. Each phase ships a real deliverable.

By Phase 6 you've stood up a SIEM, hunted for backdoors, written a custom detection rule, and produced the SOC 2 evidence package the CISO actually needs.

1
🛡️ Stand up the SIEM
Bootstrap deploys everything in the background. Mateo briefs the case, walks Wazuh architecture, previews the MCP. You log in, tour the dashboard, run your first query.
2
🎯 Baseline the environment
Generate four MITRE TTPs on dev-server-01 matching the IR report's pattern classes. Investigate the chain manually. Confirm SIEM coverage before you start hunting.
3
🔗 Threat-model the MCP
Three concrete failure modes + mitigations (stolen JWT, prompt injection, supply chain). Re-run the baseline through plain English. AI-drafted CISO update, human-verified.
4
🎯 The backdoor hunt
Four structured hunts against the three persistence categories (account, listener, scheduler) plus an AI-verification drill. The hunt log is SOC 2 evidence.
5
⚡ Tripwires and response
Write custom rule 100001 — the CloudVault client-data tripwire Dana asked for. Validate with wazuh-logtest. Deploy, trigger, verify. Take an automatic active response.
6
🧹 Close the case
End-to-end IR on a fresh alert. Evidence package for Dana and the SOC 2 audit (CC7.1 / CC7.2). A real personal artifact for interviews. terraform destroy with verification.

Quick start

Five steps. The first one is the only one you'll think about.

Full prerequisites checklist is in the README — AWS account + CLI configured, Terraform ≥ 1.5, an EC2 key pair, Claude Code CLI.

# 1. Clone the repo
git clone https://github.com/botz-pillar/wazuh-ai-soc-lab.git
cd wazuh-ai-soc-lab

# 2. Configure (your IP + your EC2 key name)
cp terraform/terraform.tfvars.example terraform/terraform.tfvars
$EDITOR terraform/terraform.tfvars

# 3. Deploy. One command. ~20 min, mostly hands-off.
./scripts/bootstrap.sh

# 4. Launch Claude Code in the lab directory
claude

# 5. Tell Mateo you're starting
> Start the lab

Mateo takes it from there. While Terraform applies and Wazuh boots, Mateo briefs you on the case, walks the architecture, and gets you ready. No dead time.
